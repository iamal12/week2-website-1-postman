What is Postman?

Postman is an API platform for building and using APIs. Postman simplifies each
step of the API lifecycle and streamlines collaboration so you can create better
APIs—faster.

API Tools

A comprehensive set of tools that help
accelerate the API lifecycle - from design,
testing, documentation and mocking to
discovery

API Repository

Easily store, iterate and collaborate around
all your API artifects on one central platform
used across teams.

Workspaces

Organize your API work and collaborate
with teammates across your organization or
stakeholders across the world

Governance

Improve the quality of APISs with governance
rules that ensure APIs are designed. built,
tested, and distrbuted meeting
organizational standards.
